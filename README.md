# Cinémathèque

Une application React pour gérer votre collection de films, séries, livres et BD.

## Installation

### Avec pnpm
```bash
pnpm install
pnpm dev
```

### Avec Bun
```bash
bun install
bun run dev
```

### Avec npm
```bash
npm install
npm run dev
```

## Scripts

- `pnpm dev` / `bun run dev` - Lance le serveur de développement
- `pnpm build` / `bun run build` - Construit l'application pour la production
- `pnpm preview` / `bun run preview` - Prévisualise la build de production

## Structure

- `src/` - Code source React
- `public/` - Fichiers statiques (CSS, manifest, etc.)
- `index.html` - Point d'entrée HTML
- `vite.config.js` - Configuration Vite
